# SmartCare Cleaning Company 🧹

> Professional. Accurate. Smart and ideal cleaning organization.

A clean, responsive website for **SmartCare**, a professional cleaning company. Built with vanilla HTML, CSS, and JavaScript.

## Tech Stack

- **HTML** – Page structure and content
- **CSS** – Styling and layout
- **JavaScript** – Interactivity and dynamic behavior

## Getting Started

No dependencies or build step required. Just clone and open:

```bash
git clone https://github.com/majidbrian/smartcare-cleaning-company.git
cd smartcare-cleaning-company
open index.html
```

## Project Structure
├── index.html   # Main page
├── style.css    # Styles
└── script.js    # Scripts
